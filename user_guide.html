دليل استخدام موقع شركة الوليد للإطارات والدواليب

الإصدار 1.0 - مايو 2025

مقدمة

مرحباً بك في دليل استخدام موقع شركة الوليد للإطارات والدواليب. يهدف هذا الدليل إلى مساعدتك في فهم هيكل الموقع وكيفية تحديثه وإدارته بسهولة.

ملاحظة: هذا الدليل مخصص للمستخدمين الذين لديهم إمكانية الوصول إلى ملفات الموقع وتحديثها. إذا كنت تحتاج إلى مساعدة إضافية، يرجى التواصل مع مطور الموقع.

هيكل الملفات

يتكون الموقع من مجموعة من الملفات والمجلدات المنظمة بالشكل التالي:

alwaleed-tires/ ├── index.html # الصفحة الرئيسية ├── products.html # صفحة المنتجات ├── services.html # صفحة الخدمات ├── about.html # صفحة من نحن ├── gallery.html # صفحة معرض الصور ├── blog.html # صفحة المدونة ├── contact.html # صفحة اتصل بنا ├── sitemap.xml # خريطة الموقع لمحركات البحث ├── robots.txt # ملف توجيهات لمحركات البحث ├── assets/ # مجلد الموارد │ ├── css/ # ملفات التنسيق │ │ └── style.css # ملف التنسيق الرئيسي │ ├── js/ # ملفات جافا سكريبت │ │ ├── main.js # ملف جافا سكريبت الرئيسي │ │ └── seo-enhancement.js # ملف تحسين محركات البحث │ └── images/ # مجلد الصور │ ├── alwaleed_logo_new.jpg # شعار الشركة │ └── ... # صور أخرى └── compatibility_test_report.html # تقرير اختبار التوافق

تحديث محتوى الموقع

تحديث النصوص

لتحديث النصوص في الموقع، اتبع الخطوات التالية:

  1. افتح ملف HTML الخاص بالصفحة التي ترغب في تحديثها باستخدام محرر نصوص (مثل Visual Studio Code، Notepad++، إلخ).
  2. ابحث عن النص الذي ترغب في تغييره.
  3. قم بتعديل النص مع الحفاظ على وسوم HTML المحيطة به.
  4. احفظ الملف.
تحذير: احرص على عدم تغيير أو حذف وسوم HTML المحيطة بالنص، حيث قد يؤدي ذلك إلى مشاكل في عرض الموقع.

تحديث الصور

لتحديث الصور في الموقع، اتبع الخطوات التالية:

  1. قم بتحضير الصورة الجديدة بنفس الأبعاد والجودة المناسبة.
  2. انسخ الصورة الجديدة إلى مجلد assets/images/.
  3. افتح ملف HTML الخاص بالصفحة التي تحتوي على الصورة.
  4. ابحث عن وسم <img> الخاص بالصورة التي ترغب في تغييرها.
  5. قم بتغيير قيمة السمة src لتشير إلى الصورة الجديدة.
  6. تأكد من تحديث قيمة السمة alt لتصف الصورة الجديدة بشكل صحيح.
  7. احفظ الملف.
<img src="assets/images/new-image.jpg" alt="وصف الصورة الجديدة">

إضافة صفحات جديدة

لإضافة صفحة جديدة إلى الموقع، اتبع الخطوات التالية:

  1. انسخ أحد ملفات HTML الموجودة (مثل about.html) واحفظه باسم جديد (مثل new-page.html).
  2. افتح الملف الجديد وقم بتحديث العناصر التالية:
    • عنوان الصفحة (وسم <title>)
    • وصف الصفحة (وسم <meta name="description">)
    • الكلمات المفتاحية (وسم <meta name="keywords">)
    • محتوى الصفحة (داخل وسم <body>)
  3. قم بتحديث القائمة الرئيسية في جميع صفحات الموقع لإضافة رابط للصفحة الجديدة.
  4. قم بتحديث ملف sitemap.xml لإضافة الصفحة الجديدة.
ملاحظة: تأكد من الحفاظ على نفس هيكل وتنسيق الصفحات الأخرى للحفاظ على تناسق الموقع.

تحديث التأثيرات الحركية

يحتوي الموقع على العديد من التأثيرات الحركية التي تم تنفيذها باستخدام CSS وJavaScript. لتعديل هذه التأثيرات:

تعديل تأثيرات CSS

  1. افتح ملف assets/css/style.css.
  2. ابحث عن القسم الخاص بالتأثير الذي ترغب في تعديله (عادة ما يكون مصحوباً بتعليق توضيحي).
  3. قم بتعديل خصائص CSS حسب الحاجة.
  4. احفظ الملف.

تعديل تأثيرات JavaScript

  1. افتح ملف assets/js/main.js.
  2. ابحث عن الدالة أو الكود الخاص بالتأثير الذي ترغب في تعديله.
  3. قم بتعديل الكود حسب الحاجة.
  4. احفظ الملف.
تحذير: قبل إجراء أي تعديلات على ملفات JavaScript، قم بعمل نسخة احتياطية من الملفات الأصلية.

تحسين محركات البحث (SEO)

تم تحسين الموقع لمحركات البحث من خلال العناصر التالية:

العناوين والأوصاف

لتحديث عنوان ووصف الصفحة:

  1. افتح ملف HTML الخاص بالصفحة.
  2. ابحث عن وسم <title> وقم بتحديثه.
  3. ابحث عن وسم <meta name="description"> وقم بتحديث قيمة السمة content.
  4. ابحث عن وسم <meta name="keywords"> وقم بتحديث قيمة السمة content.

خريطة الموقع

لتحديث خريطة الموقع:

  1. افتح ملف sitemap.xml.
  2. أضف أو قم بتحديث عناصر <url> لكل صفحة في الموقع.
  3. تأكد من تحديث تاريخ آخر تعديل <lastmod> لكل صفحة تم تحديثها.

البيانات الهيكلية

لتحديث البيانات الهيكلية (Schema Markup):

  1. افتح ملف assets/js/seo-enhancement.js.
  2. ابحث عن الكائنات JSON الخاصة بالبيانات الهيكلية وقم بتحديثها حسب الحاجة.

نشر التحديثات

بعد إجراء التحديثات على الموقع، اتبع الخطوات التالية لنشرها:

  1. قم بمراجعة جميع التغييرات للتأكد من عدم وجود أخطاء.
  2. اختبر الموقع محلياً للتأكد من عمل جميع الصفحات والتأثيرات بشكل صحيح.
  3. قم بنسخ جميع الملفات المحدثة إلى خادم الاستضافة باستخدام برنامج FTP أو لوحة تحكم الاستضافة.
  4. تحقق من الموقع المنشور للتأكد من ظهور التحديثات بشكل صحيح.
ملاحظة: يُفضل عمل نسخة احتياطية من الموقع قبل نشر أي تحديثات كبيرة.
